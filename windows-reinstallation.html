Windows Reinstallation Guide

Posted on 2020-07-13 Edited on 2024-09-12 In Windows Word count in article: 2.2k Reading time ≈ 8 mins.
Tutorial on Reinstalling the Windows Operating System, Along with Installation Instructions for Office

This post was translated from my Chinese blog post with the aid of ChatGpt.

Tools

  • One USB flash drive with a capacity of ≥8GB.
  • One functional computer.
  • Rufus bootable USB creation tool.
    • Download Link
    • On the website, under the Download section, select Rufus x.xx Portable.
  • Windows ISO file.
    • For Windows 10 versions 1909 and earlier:
      • Download Link
      • Click on the Operating Systems in the left sidebar.
      • Choose the version you want to install.
      • For example, using Windows 10, Version 1909 (updated Jan 2020) as an example:
        • 1909 indicates that this is the version from September 2019 (the next version would be 2004, meaning the version released in April 2020).
        • updated Jan 2020 indicates that this image includes small updates from January 2020.
      • You will see four options; for example, choose Windows 10 (business editions), version 1909 (updated Jan 2020) (x64) - DVD (Chinese-Simplified).
        • Business editions is the version we need, which includes Enterprise, Professional, Education, and Professional Workstation editions, among others.
        • x64 signifies the 64-bit system (32-bit systems are marked as x86). It is recommended to install the 64-bit version unless you have specific requirements for the 32-bit version.
      • Click on Details, and you will find a link starting with ed2k.
      • Copy this link and use a download manager (e.g., Thunder) to download the image file.
    • For Windows 10 versions 2004 and later, including Windows 11:
      • Download Link
      • Register and log in using your email or third-party account.
      • If you can, consider donating to support the website; the author has been providing this service for free for 12 years.
      • Select Windows10 and click on Details.
      • For example, use Windows 10 (business edition), Version 2004 (Updated June 2020) (x64) - DVD (Chinese-Simplified) as an example.
      • Click the Copy button next to it.
      • Open your preferred download software (e.g., Motrix) and start downloading the image.

Creating System Installation Media

  • Insert the USB drive.
  • Run Rufus with administrative privileges.
  • In the Device section, choose your USB drive.
  • For Boot selection, keep the default Disk or ISO image (Please select), then click the adjacent Select button.
  • Locate and open the downloaded Windows 10 image.
  • Click the Start button at the bottom.
  • Wait for the creation process to finish.

Modifying BIOS Boot Settings

The following options may vary; refer to your computer’s BIOS documentation for accuracy. The general idea is to:

  • Restart your computer.
  • During startup, repeatedly press F2/F10/Delete (the key to enter BIOS varies by computer).
  • Enter the BIOS.
  • Find Boot order/Boot item management.
  • Disable Secure Boot.
  • For new computers:
    • Select UEFI Only for boot mode.
    • Set your USB drive as the First boot item (some computers require pressing F5/F6 to move it to the top).
    • If there are two USB drive options, e.g., UEFI:USBXXX and USBXXX, choose the one with UEFI.
    • Save settings and exit the BIOS.
  • For older computers (not supporting UEFI boot):
    • Set your USB drive as the First boot item (some computers require pressing F5/F6 to move it to the top).
    • Save settings and exit the BIOS.
  • Wait for the computer to restart.

Converting Hard Drive Partition Format

This step is only for UEFI boot. If not using UEFI, please skip.

  • After the computer finishes its startup cycle, you will reach the language selection screen.
  • Press Shift + F10 to open a terminal.
  • Type diskpart to open the disk management utility.
  • Enter list disk to list available disks, displaying their numbers (0/1/2, etc.).
  • Identify the target installation disk by its number and enter select disk number (be cautious not to select your USB drive).
  • Type clean to wipe the disk.
  • Enter convert gpt to convert it to the GPT format.
  • Re-enter list disk and verify if there is an asterisk under GPT.
  • If there is an asterisk, you can close the terminal.

Installation and Configuration

The installation process may vary slightly for different versions of Windows 10, and not all of the following steps will be the same on all systems:

  • After selecting your installation language, click Next.
  • Click Install Now.
  • On the Activate Windows page, select I don't have a product key.
  • On the Select the operating system you want to install page, choose the version you want to install and click Next.
  • On the Applicable notices and license terms page, check I accept the license terms and click Next.
  • On the Which type of installation do you want? page, select Custom: Install Windows only (advanced).
  • On the Where do you want to install Windows? page, determine your hard drive based on its size. If you have a solid-state drive, choose it.
  • Click New at the bottom right; this will prompt you to create additional partitions. Click OK.
  • After creating the partitions, click Next.
  • Wait for the system to complete the installation; this step may take some time.
  • When you see the prompt Windows needs to restart to continue, with a 10-second countdown, remove the USB drive and click Restart now.
  • After restarting, you’ll be prompted to start with regional settings. Choose China and click Yes.
  • You’ll be asked, Is this the right keyboard layout? Select your preferred keyboard layout and click Yes.
  • If needed, you can add a second keyboard layout; the steps are the same. If not, click Skip.
  • Connect to the internet. If you don’t want to or can’t connect, click I don't have Internet.
    • If you choose not to connect, you’ll see a second prompt: There's more to discover when you're connected to the internet. Click Continue with limited setup in the lower-left corner.
  • Enter your username and password and confirm your password.
  • The three security questions are optional; you can provide random answers since they’re rarely used.
  • On the Do more across devices with activity history page, choose Yes if you have multiple computers to sync activity history across. However, it’s not particularly useful for most users.
  • On the Get help from your digital assistant page, select Accept to enable Cortana, although it’s not very useful.
  • On the Choose privacy settings for your device page, make selections based on your preferences. Personally, I only enable Location and Find my device.
  • After clicking Accept, the setup is complete. Wait a moment, and you’ll be taken to the desktop.

Updates

After reaching the desktop, it’s recommended to connect to the internet. Then, go to Settings > Update & Security > Windows Update and wait for all updates to install. Once they’re installed, click Restart now. This process may take a while depending on your internet speed. Additionally, Windows 10 will automatically install drivers in the background, so if your screen resolution seems odd at first, don’t worry – it will be corrected after the drivers are installed.

KMS Activation

KMS, short for Key Management Service, is primarily used for bulk activation of computers within enterprises. The activation period for KMS is 180 days, but it can be set to auto-renew. It’s important to note that, from a legal perspective, KMS used by individual users does not constitute genuine licensing. Genuine licensing requires purchasing a key from the official Microsoft website. Microsoft tends to turn a blind eye to this practice, but it’s not advisable for long-term use.

Office Installation and Activation

Installation Method:

  • Download Office Tool Plus, it is recommended to download the complete version.
  • Extract the files and run Office Tool Plus.exe as an administrator.
  • Click on Deployment.
  • In the Product section, select the version you need.
    • Typically, you would choose Office 专业增强版 2019 批量版 ProPlus2019Volume.
  • In the Applications section, select the software you need.
  • Choose the x64 option in the architecture on the right side. For 32-bit systems, select x86.
  • Select the appropriate channel.
    • Usually, 每月通道 (Monthly Channel) is selected.
    • Enterprise versions of Windows 10 might prompt that the channel is not supported. In this case, you can switch to Office 2019 企业长期版 (Office 2019 Enterprise Long Term Servicing Channel).
    • Alternatively, you can initially choose 测试通道 (Insider Channel) or 开发通道 (Dev Channel) and then switch back to 每月通道 to avoid errors.
  • Keep the installation method as the default 在线安装 (Online Installation).
  • Keep the installation module as the default Office部署工具 (Office Deployment Tool).
  • Check 安装完成后创建桌面快捷方式 (Create desktop shortcut after installation).
  • Click 开始部署 (Start Deployment) in the lower right corner.

You can also activate Office using KMS activation with the following steps:

  • Run Office Tool Plus.exe as an administrator.
  • Click on 激活 (Activate).
  • In KMS管理 (KMS Management), enter the KMS server address as kms.03k.org and click 应用服务器地址 (Apply server address).
  • In 密钥管理 (Key Management), enter the appropriate version’s product key and click 安装密钥 (Install key).
  • Click 激活 (Activate) in the lower right corner.
